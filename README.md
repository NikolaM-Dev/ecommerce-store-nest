# Description

`Rest API` to learn nestjs professionally

![api/docs](./assets/images/api_docs.png)
![endpoint](./assets/images/endpoint.png)

## Installation

```bash
# yarn
yarn

# npm
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
